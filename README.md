# vueclass

## CLI 설치하기

npm install -g @vue/cli

## VUE 설치하기

vue create vueclass

## firebase CLI 설치

sudo npm install -g firebase-tools

## firebase CLI 에서 로그인

firebase login

## firebase CLI 초기화

firebase init
